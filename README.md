# 개발자 키보드 세팅 파일


## Karabiner.json
From | To
---|:---:
cmd + q | Back Space  
cmd + e | Delete  
cmd + y, cmd + o | 단위 이동
cmd + u, cmd + i | Home, End
cmd + p, cmd + ; | Page Up, Page Down
cmd + h j k l | arrow left, up, down, right
right_cmd | F13 (한영 변환)
right_option | F13 (한영 변환)

### Location : /Users/\<User Name\>/.config/karabiner/karabiner.json
